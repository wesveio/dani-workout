  <how-to-verify>
1. Open the preview URL (http://localhost:4173) on your phone or in Chrome DevTools mobile view (375px width)
2. **Theme check:** Verify dark background (#0D0D0D), red accent buttons, orange secondary elements. Cards should be slightly lighter (#1A1A1A) than the background.
3. **Navigation check:** Bottom nav has 5 tabs: Inicio, Treino, Historico, Templates, Config. Tap each one. Active tab should show white icon with small red dot below. Templates tab should show "Em breve" placeholder.
4. **Header check:** App name "Dani" in large text on left, profile switcher on right. No program info clutter.
5. **Timer check:** Navigate to a workout session. Mark a set as completed. Timer should auto-start immediately with a floating card showing the orange ring countdown. Watch the ring deplete clockwise.
6. **Urgency check:** Wait for timer to reach 10 seconds. Ring color should switch from orange to red.
7. **Expiry check:** Let timer expire. You should hear a short two-tone chime. Card should show "Pronto!" then auto-dismiss after 2 seconds.
8. **Skip check:** Start another set timer, then tap "Pular Descanso". Timer should dismiss immediately.
9. **Rest config check:** Tap the gear icon on any exercise card. A sheet should open with 30s, 60s, 90s presets and a "Personalizado" option. Select a different duration, confirm, then complete a set -- timer should use the new duration.
10. **Overall feel:** Does it look like a premium gym app? Bold but readable?
