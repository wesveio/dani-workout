## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `#0D0D0D` | Page background, deep surface behind radial gradients |
| Secondary (30%) | `#1A1A1A` | Cards, session exercise cards, nav pill background, settings sheet |
| Accent (10%) | `#FF3D3D` | See reserved list below |
| Accent Secondary | `#FF8C00` | Timer ring fill color (active countdown arc only) |
| Muted | `#888888` | Secondary text, placeholder text, previous-session values |
| Neutral | `#2A2A2A` | Borders, dividers, input outlines, inactive nav items |
| Foreground | `#F5F5F3` | Primary text, headings, active nav icon |
| Destructive | `#EF4444` | Delete profile action, destructive confirmations only |

**Accent (`#FF3D3D`) reserved for:**
1. Primary action buttons (Save Set, Start Workout, Confirm)
2. Active/selected nav tab indicator dot or icon
3. PR badge background
4. Timer ring track fill when <10 seconds remaining (urgency signal)
5. Toggle/switch active state

**Accent is NOT used for:** General text, borders, backgrounds, icon fills on non-active states, the timer ring when >10 seconds remain (use `#FF8C00` for normal countdown).

**Radial gradient update (index.css body):**
Replace existing green/blue radial gradients with:
```
radial-gradient(circle at 20% 20%, rgba(255,61,61,0.04), transparent 35%),
radial-gradient(circle at 80% 0%, rgba(255,140,0,0.05), transparent 30%),
#0D0D0D
```

## Rest Timer Visual Contract

**Placement:** Floating card, `fixed bottom-[88px]` (above session footer), centered, `w-[90vw] max-w-sm`, `z-30`.

**Card anatomy:**
- Background: secondary `#1A1A1A`, `rounded-xl` (24px), `shadow-soft`
- Ring: 120×120px SVG, `r=44`, track stroke `rgba(255,255,255,0.1)` width 6, progress arc `#FF8C00` width 6
- Ring urgency: when remaining ≤ 10s, arc stroke switches to `#FF3D3D`
- Center of ring: countdown numerals in 36px/700/line-height-1.0, foreground color
- Below ring: rest label in 14px/500 muted text (e.g., "Descanso")
- Skip button: `ghost` variant, 44px touch target, positioned bottom-right of card, label "Pular"
- Ring animation: CSS `transition: stroke-dashoffset 0.5s linear` on progress arc
- On `visibilitychange` return: disable transition class, apply corrected offset, re-enable in next `requestAnimationFrame`
- Clockwise depletion: SVG rotated `-90deg` so 12 o'clock is start position

**States:**
1. **Active countdown:** Ring depleting, seconds ticking, Skip visible
2. **Urgency (<= 10s):** Arc color switches to accent red `#FF3D3D`
3. **Rest complete:** Card shows "Pronto!" in heading/20px, no ring, auto-dismisses after 2s or tap
4. **Expired while backgrounded:** On foreground return, immediately show "Pronto!" state (no countdown)

Source: CONTEXT.md D-08 (Claude's discretion — floating card chosen), D-09 (clockwise ring), D-11, D-13, RESEARCH.md §Pattern 1.

## Interaction Contract

### Set Completion Flow
1. User taps set checkmark in SessionDetail
2. Set state transitions to `completed: true`
3. `startRestTimer(exerciseRestSeconds)` called immediately — no prompt (D-11)
4. `RestTimerCard` appears with full ring and countdown
5. Timer counts down; audio chime + vibration fires at zero (D-12)
6. Card transitions to "Pronto!" state, auto-dismisses after 2s

### Per-Exercise Rest Config
- Entry point: gear icon (Lucide `Settings2`, 20px) on each exercise card header in SessionDetail
- Opens `ExerciseRestSheet` (shadcn Dialog as bottom sheet)
- Options: pill buttons for "30s", "60s", "90s", "Custom"
- Custom: numeric input (seconds), 44px height, max 600s enforced
- Default global: 90s applied when no per-exercise override exists
- Persisted to workoutStore

### Two-Tap Rule Verification (D-07)
- Dashboard → any major screen: 1 tap (bottom nav)
- Rest config: 2 taps (exercise card gear → sheet)
- All satisfied within the 5-tab nav structure

## Animations

| Animation | Trigger | Spec |
|-----------|---------|------|
| `celebration-pop` | PR badge appears | Existing — keep unchanged |
| `float-spark` | Set completion spark | Existing — update spark color to `#FF3D3D` |
| Timer ring depletion | Every 500ms tick | CSS `transition: stroke-dashoffset 0.5s linear` |
| Timer card entrance | Timer starts | `celebration-pop` variant: scale 0.9→1.0, translateY 8px→0, opacity 0→1, 200ms ease-out |
| Timer card exit | Skip or complete | opacity 1→0, scale 1.0→0.95, 150ms ease-in |
| Urgency pulse | remaining ≤ 10s | `ring-urgency` keyframe: arc color cross-fade over 300ms from `#FF8C00` to `#FF3D3D`, no pulse/shake |

`prefers-reduced-motion`: all transitions and animations collapse to `0.01ms` — existing rule in index.css already handles this.
