    Component with internal visibility state initialized on mount (not render — per Pitfall 2):

    ```tsx
    import { useState, useEffect } from 'react'
    import { Share2 } from 'lucide-react'
    import { shouldShowBanner, dismissBanner } from '@/lib/iosInstall'
    import { Button } from '@/components/ui/button'

    export function iOSInstallBanner() {
      const [visible, setVisible] = useState(false)

      useEffect(() => {
        // Check on mount only, not every render (Pitfall 2)
        const timer = setTimeout(() => {
          setVisible(shouldShowBanner())
        }, 3000) // Show after 3 seconds per UI-SPEC
        return () => clearTimeout(timer)
      }, [])

      if (!visible) return null

      return (
        <div
          className="fixed bottom-[72px] left-0 right-0 mx-auto flex max-w-[360px] items-center gap-3 rounded-xl bg-surface border border-neutral/50 px-4 py-3 shadow-soft animate-in slide-in-from-bottom-4 duration-200"
          style={{ borderLeft: '2px solid #FF8C00' }}
          role="banner"
        >
          <Share2 className="h-5 w-5 shrink-0 text-muted" />
          <span className="flex-1 text-sm">Adicione ao inicio para acesso rapido</span>
          <Button
            variant="ghost"
            size="sm"
            className="text-muted shrink-0"
            onClick={() => {
              dismissBanner()
              setVisible(false)
            }}
          >
            Dispensar
          </Button>
        </div>
      )
    }
    ```

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Camera/gallery file -> IndexedDB | User-selected photo file enters persistent storage via compression |
| Import JSON with photos -> IndexedDB | base64 photo data in untrusted bundle crosses into database |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-06-07 | Tampering | PhotoGallery file input | mitigate | accept="image/*" restricts MIME type. compressImage outputs JPEG only. dataUrl rendered only via <img src={}>, never innerHTML. |
| T-06-08 | Tampering | importData progressPhotos | mitigate | Zod progressPhotoSchema validates id, userId, date, dataUrl (string min 1), fileSizeBytes (number). Malformed entries rejected. |
| T-06-09 | Denial of Service | Photo storage exhaustion | mitigate | 1MB hard cap per photo. navigator.storage.estimate() warns at 80% quota. QuotaExceededError caught in store with user-facing message. |
| T-06-10 | Spoofing | iOSInstallBanner | accept | UA sniffing is best-effort. Worst case: banner shows on non-iOS or doesn't show on iOS. No security impact. |
</threat_model>
